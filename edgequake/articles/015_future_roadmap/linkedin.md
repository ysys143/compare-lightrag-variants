# What's Next for EdgeQuake? The 2025 Roadmap

Open source projects die in silence. Roadmaps prove we're building for the long term.

Here's where EdgeQuake is headed:

## 🚀 Q1 2025: Foundation

**Anthropic Claude Integration**
→ Native adapter for Claude 3 (Opus, Sonnet, Haiku)
→ 200K context window for large documents
→ Cost tracking included

**Python SDK**
→ Sync and async APIs
→ Pandas/Polars integration
→ PyPI package with minimal dependencies
→ 70% of ML teams work in Python—we're meeting you there

**CLI Tool**
→ `eq ingest`, `eq query`, `eq status`
→ Unix pipeline support
→ Developer experience matters

**OpenTelemetry Observability**
→ Distributed tracing
→ Prometheus metrics
→ Grafana dashboards

## 🏢 Q2 2025: Enterprise

**SSO (OIDC/SAML)**
→ Okta, Auth0, Azure AD, Google
→ Group-to-role mapping
→ No more separate credentials

**RBAC (Role-Based Access Control)**
→ Admin, Editor, Viewer roles
→ Custom role definitions
→ API key scoping

**Document Formats**
→ Microsoft Office (DOCX, XLSX, PPTX)
→ HTML, Email (EML, MSG)

**Audit Export**
→ JSON export for SIEM
→ Splunk/Datadog connectors

## 🔮 Q3-Q4 2025: Vision

**AI Agents**
→ Multi-turn conversations with memory
→ Tool use and external APIs
→ Workflow automation

**Multi-hop Reasoning**
→ Follow relationship chains
→ Explain reasoning paths

**Graph Embeddings**
→ Node2Vec, GraphSAGE
→ Combined vector+graph search

**LangChain/LlamaIndex SDKs**
→ Official integrations
→ Ecosystem compatibility
