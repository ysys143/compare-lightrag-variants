# Academic Paper

## Abstract

This paper examines the efficiency of PDF extraction algorithms. We present a novel approach to handling multi-column layouts with improved accuracy.

## 1. Introduction

PDF extraction is a crucial task in document processing. Previous work has focused on single-column documents, leaving gaps in multi-column handling.

## 2. Method

Our approach uses histogram projection combined with adaptive thresholding.

## 3. Results

We achieved 94% accuracy on two-column layouts and 89% on three-column layouts.

| Metric    | Score |
| --------- | ----- |
| Accuracy  | 94%   |
| Precision | 92%   |
| Recall    | 96%   |

## 4. Conclusion

This work advances the state of PDF extraction for complex layouts.
