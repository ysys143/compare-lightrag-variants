# Mathematical Formulas

## Inline Math

The equation $x^2 + y^2 = z^2$ represents the Pythagorean theorem.

Using the quadratic formula: $x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$

## Block Math Formulas

Euler's identity:
$$e^{i\pi} + 1 = 0$$

Integral:
$$\int_0^{\infty} e^{-x} dx = 1$$
