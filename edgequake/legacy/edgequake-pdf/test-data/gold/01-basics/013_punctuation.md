# Punctuation Handling

Testing different punctuation marks in text:

- Question mark: Are you ready?
- Exclamation: This is amazing!
- Ellipsis: Wait...
- Dashes: This—is—a—test
- Hyphens: This-is-hyphenated
- Commas: One, two, three, four
- Semicolons; Used in; Lists or; Clauses
- Apostrophes: It's the editor's choice
- Quotation: "Double quotes" and 'single quotes'
- Parentheses: This (is important) in text.
- Brackets: This [is a note] in brackets.
- Braces: This {is style} in braces.
