# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in EdgeQuake, please report it privately by emailing the maintainers at the address listed in the repository or by opening a security advisory on GitHub. Do not disclose security issues publicly until they have been reviewed and patched.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1   | :x:                |

## Security Updates

Security updates will be released as needed. Please ensure you are using the latest version.
